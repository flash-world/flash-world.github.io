About ALA MC

ALA MC is a Vanilla Minecraft Survival Server aimed for compatibility on low end devices while still allowing an overall great for everyone. The server does not allow cracked accounts at the moment. We are currently running Java 1.8.9

The server is being hosted on a Lenovo ThinkStation E30 running Ubuntu. Below is the server ip.

Rules

Failure to comply may result in a kick or ban

  1. NO Hacking! This also includes xray and any exploitation of a bug(e.g. duping)

    This includes using clients like Wurst. We are not like other servers and duping is strictly not allowed.

  2. Do not try to crash the server in any way

    Lag Machines and such will have punishment. Running a server costs resources even if I'm doing it myself.

  3. This is NOT an anarchy server

    There are rules but there are still many freedoms on this server

  4. No random griefing

    You are allowed to attack other players but if you are on the server ruthlessly griefing and not really playing then you will be punished.

  5. No Offline raiding

    You are not allowed to raid or attack a base if the person or majority of the people who are a part of the base are offline

Basically just have some common sense. Have fun!

Short guide to playing on ChromeBook/Mobile

You can play Minecraft: Java Edition on a ChromeBook or a Mobile device. You can do this by installing pojavlauncher on the Play Store. Pojavlauncher is also availible for iOS and Android. Pojavlauncher allows use of Microsoft accounts and cracked accounts. I am not promoting the use of cracked accounts. You are responsible for such action.

About

The server uses the dynmap plugin allowing for a real-time view of the server map. You can see the locations of players in real time. There is a isometric view of the map availible. Along with that you can chat with players on the server.

What determines if you are seen on the map?

You are seen on the map unless you are crouching, under something or in the dark.
