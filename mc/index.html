About

FlashWorldMC is a custom modded Minecraft server. Unlike other modded Minecraft servers, this modpack is focused around variety. Which means that this modpack is open to different play styles allowing you to enjoy the server the way you want.

Server IP

Since May 31st, 2025 the server port has been changed to the default (25565)

Download Modpack

The Server Modpack is a Curseforge Format modpack which means it can only be used with the CurseForge App or Prism Launcher. Also please note, This server is not cracked. You must have a legitimate Minecraft Account in order to play.


Rules

  1. NO Hacking! This also includes xray and any exploitation of a bug(e.g. duping)

    This includes using clients like Wurst. We are not like other servers and duping is strictly not allowed.

  2. Do not try to crash the server in any way

    Lag Machines and such will have punishment. Running a server costs resources.

  3. This is NOT an anarchy server

    There are rules but there are still many freedoms on this server

  4. No random griefing

    You are allowed to attack other players but if you are on the server ruthlessly griefing and not really playing then you will be punished.

  5. No Offline raiding

    You are not allowed to raid or attack a base if the person or majority of the people who are a part of the base are offline
